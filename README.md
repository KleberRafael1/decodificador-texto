# 🔐 Decodificador de Texto - Cifra de César

Aplicação web para **criptografar e descriptografar textos** utilizando a técnica clássica de criptografia chamada **Cifra de César**.

## 🌐 Deploy

Acesse o projeto online:

https://decodificador-texto-psi.vercel.app/

## 💻 Sobre o projeto

Este projeto foi desenvolvido como exercício prático para trabalhar:

- manipulação de texto
- lógica de programação
- interação com o usuário no navegador

A aplicação permite que o usuário insira um texto e escolha entre **criptografar ou descriptografar** utilizando a técnica da Cifra de César.

## 🔑 Como funciona a Cifra de César

A Cifra de César é um método de criptografia simples onde cada letra do texto é deslocada um número fixo de posições no alfabeto.

Exemplo:

Texto original:

HELLO


Texto criptografado (deslocamento +3):

KHOOR


## 🎯 Funcionalidades

- Criptografar texto
- Descriptografar texto
- Interface simples e interativa
- Execução diretamente no navegador
